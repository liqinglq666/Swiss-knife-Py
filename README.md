这是一个为了拯救“琐事焦虑症”而诞生的个人开源项目。它基于 Python 后端与极简主义前端，旨在用最优雅的拖拽动作，解决掉那些让你想砸键盘的日常办公小麻烦。

---

## ✨ 为什么叫“瑞士军刀”？

因为它就像你口袋里那把折叠刀：平时安安静静，需要时能裁纸、能开瓶、还能拧螺丝。

### 🚀 目前已解锁的神技：

* **🖼️ 论文/文档图片一键提取 (DocExtractor)**
    * **痛点**：PDF/Word 里的高清实验图想单独保存？截图质量太渣，一张张右键保存能点到手软。
    * **绝招**：直接拖进去，系统自动扫描底层流数据，将所有原始分辨率图片打包成 `.zip` 供你下载。
* **📊 跨格式表格魔法转换 (TableConvert)**
    * **痛点**：网页、PDF 里的统计表想弄进 Excel？手动敲数据是对工程师智商的侮辱。
    * **绝招**：自动结构化识别，PDF/CSV 秒变标准 `.xlsx`，保留行列逻辑。
* **印 批量隐私脱敏水印 (Watermark)**
    * **痛点**：发个证件照怕被二次挪用？手动用美图秀秀加水印太“业余”。
    * **绝招**：全屏 45° 倾斜半透明水印，保护隐私，拒绝白嫖。

## 🎨 视觉进化：从“上世纪”到“现代极简”

我们彻底告别了那些花哨且低效的布局。本项目采用了 **Minimalism (极简主义)** 风格：

* **配色方案**：微软 Azure 蓝 + 高级灰 + 纯白底色，减少视觉干扰。
* **交互逻辑**：左侧控制参数，右侧核心拖拽，中间留白呼吸感十足。
* **图标支撑**：全面引入 **FontAwesome 6.0** 专业图标库，质感直接拉满。

## 🛠️ 快速启动 (给还没装环境的兄弟)

1.  **克隆项目**:
    ```bash
    git clone [https://github.com/liqinglq666/Swiss-Knife-Py.git](https://github.com/liqinglq666/Swiss-Knife-Py.git)
    cd Swiss-Knife-Py
    ```

2.  **喂饱你的 Python 环境**:
    ```bash
    pip install -r requirements.txt
    ```

3.  **点火，启动！**:
    ```bash
    python app.py
    ```

4.  **开始表演**: 浏览器打开 `http://127.0.0.1:5000`，剩下的交给鼠标拖拽。
